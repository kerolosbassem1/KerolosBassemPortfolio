# Kerolos Bassem Portfolio

Personal portfolio website for Kerolos Bassem Fekry, a frontend engineer based in Cairo, Egypt.

Built with Vue 3 and Vite. The site includes a hero section, CV-backed about content, skills, projects, experience, contact links, and a downloadable CV.

## Project Setup

```sh
npm install
```

## Development

```sh
npm run dev
```

## Production Build

```sh
npm run build
```
